【変数使用状況】
CALLMAUD モードの主人に対する呼び方
MLFCOM リリーゼ版FCOM

MAUD:0 = 前の調教で破瓜を迎えたかどうか 日付が入る
MAUD:1 = 昼に体力切れしたフラグ
MAUD:2 = 最後に調教した日
MAUD:3 = 裁縫イベントに使用
MAUD:4 = オナホの汚れ ビット演算で0がモード、1が主人、2がリリーゼ
;MAUD:5 = モードとの子の性別 ランダムでビット演算
MAUD:6 = リリーゼ助手時に調教されたかどうか
MAUD:7 = 添い寝イベントの確認
MAUD:8 = リリーゼ「あたしの知ってるおっぱいじゃない……」 リリーゼが想像してるモードのバストサイズ
MAUD:9 = MAUD:8を使用したイベントを見たかどうか
MAUD:10 = レーヴァテインに舐められたかどうか
MAUD:11 = 某イベントで使用
MAUD:12 = リリーゼがモードのAV見たかどうか
MAUD:13 = モードとリリーゼが共演したアダルトビデオを持ってるかどうか
MAUD:14 = ニプルキャップ付けてから何ターン経過したか
MAUD:15 = リリーゼが竿役でモードとセックスしたかどうか
;MAUD:16 = リリーゼがオナホ経験済みかどうか 現状の仕様だとリリーゼ口上書かないとちょと難しい

MAUD:50 = 1月のあけましておめでとうイベントに使用 一年に一回だけなのでイベント発生年が入る
MAUD:51 = バレンタインイベントに使用 中身は同上
MAUD:52 = クリスマスイベントに使用 中身は同上
MAUD:98 = リリーゼが陥落済みでMAUD:99が1になってたら1 要はレーヴァテインが喋るかどうかフラグ
MAUD:99 = ドラゴンたちのセリフを表示するかどうかのフラグ

HERE_L = リリーゼが助手かどうか 調教開始時に判別
