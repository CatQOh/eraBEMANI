
eraCharaViewer
作者: 緑茶
ライセンス: CC0

■概要
過去 eramakerViewer という、キャラデータなどの CSV ファイルを HTML 形式で出力するツールがありました。
IRC で eramakerViewer の代わりとなるツールを作ろうという話になり、
アプリ上でのテキスト表示なら作れそうなので、作ってみました。

キャラ情報を CSV で直接見るよりは、ちょっと便利です。

■使い方
eraCharaViewer.exe と同じフォルダに CSVファイル（またはフォルダ）を全部コピーして、
eraCharaViewer.exe を実行してください

eraCharaViewer.exe と同じフォルダに TraditionalEra.dll も必要です

■サポート
サポートはありません。
エラーが発生したら、ソースを見て直してね

でも、どうしてもって言うなら、
IRC(irc.nurs.or.jp) の #eraEVENT_KXX に、よくいるので相談にのります
